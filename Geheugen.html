Het geheugen


Neunmann Architectuur

De Neumannarchitectuur, ook wel de von Neumann-architectuur genoemd, is het basismodel voor hoe computers werken, zoals we ze vandaag de dag kennen. Dit idee komt van de wetenschapper John von Neumann, die het in de jaren 1940 bedacht. Het belangrijkste aan deze architectuur is dat de computer hetzelfde geheugen gebruikt voor zowel de programma's (de instructies die zeggen wat de computer moet doen) als de data (de informatie waarmee de computer werkt). Dit betekent dat alles in één grote opslagruimte zit en de computer die dingen op dezelfde manier kan ophalen.

De computer voert taken stap voor stap uit. Eerst haalt hij een instructie op uit het geheugen, dan begrijpt hij wat hij moet doen, en vervolgens voert hij die instructie uit. Dit gebeurt steeds opnieuw en steeds maar één instructie tegelijk. Binnen de computer zijn er ook onderdelen die voor specifieke taken verantwoordelijk zijn. Zo heb je bijvoorbeeld een rekeneenheid die zorgt voor alle berekeningen en logische bewerkingen, en een controle-eenheid die alles aanstuurt.

Daarnaast zijn er de onderdelen voor invoer en uitvoer, zoals toetsenborden en beeldschermen, waarmee de computer kan communiceren met de buitenwereld. Een belangrijk aspect van deze architectuur is dat programma’s opgeslagen zijn als gegevens in het geheugen, waardoor ze gemakkelijk kunnen worden aangepast en uitgevoerd, zonder dat je de hardware hoeft te veranderen.

Een nadeel van dit systeem is dat er een soort snelheidslimiet zit op de communicatie tussen de processor en het geheugen. Omdat zowel de programma's als de data via hetzelfde kanaal naar de processor moeten, kan het soms traag worden als de computer steeds moet wisselen tussen het ophalen van instructies en het ophalen van gegevens. Dit wordt de von Neumann-bottleneck genoemd.

In het kort is de von Neumann-architectuur dus een simpel maar krachtig model dat beschrijft hoe computers informatie verwerken en instructies uitvoeren.


CPU


De CPU, oftewel de Central Processing Unit, is het belangrijkste onderdeel van een computer dat zorgt voor het uitvoeren van allerlei opdrachten. Je kunt het eigenlijk vergelijken met het brein van de computer. Alles wat je op een computer doet, zoals een programma draaien of een website laden, wordt aangestuurd en verwerkt door de CPU.

Hoe snel een CPU werkt, hangt af van de kloksnelheid, die in gigahertz (GHz) wordt gemeten. Hoe hoger dat getal, hoe meer taken de CPU per seconde kan uitvoeren. Moderne CPU’s hebben vaak meerdere kernen, wat betekent dat ze meerdere dingen tegelijk kunnen doen. Bijvoorbeeld, een CPU met vier kernen (quad-core) kan vier verschillende taken tegelijkertijd uitvoeren, wat de snelheid en efficiëntie van je computer verhoogt.

Daarnaast heeft de CPU een klein stukje supersnel geheugen, de cache, waarmee het snel gegevens kan ophalen die het vaak nodig heeft. Hierdoor hoeft de CPU niet elke keer alles vanuit het langzamere werkgeheugen te halen, wat zorgt voor een snellere werking.

Kort gezegd is de CPU dus het onderdeel dat alle rekenwerk en aansturing van de computer voor zijn rekening neemt, waardoor je computer soepel en snel kan functioneren.


RAM


RAM is het geheugen van je computer dat wordt gebruikt om dingen tijdelijk op te slaan terwijl je ermee bezig bent. Het helpt de computer om snel te werken. Als je bijvoorbeeld een spel speelt of een programma gebruikt, worden de gegevens in RAM gezet, zodat de computer snel toegang heeft tot wat hij nodig heeft.

Het is snel, maar niet blijvend. Zodra je de computer uitzet, verdwijnt alles wat in het RAM stond. Hoe meer RAM je hebt, hoe beter je computer meerdere taken tegelijk kan doen, zonder dat hij traag wordt.
